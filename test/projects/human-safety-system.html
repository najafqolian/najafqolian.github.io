Human Safety System

Project Overview

Developed a Human Safety System for industrial environments, using wearable technology to ensure worker safety. The system alerts workers when they enter dangerous zones and can automatically shut down hazardous machinery in emergencies.

Key Features:

  • Accurate danger zone detection using proximity sensors.
  • Real-time alarms for workers in unsafe areas.
  • Integration with machinery for automated shutdown.
