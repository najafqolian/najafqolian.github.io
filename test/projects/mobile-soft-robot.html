Mobile Soft Robot

Project Overview

Developed a Mobile Soft Robot designed for agricultural tasks, such as fruit picking. Equipped with machine vision and a soft gripper, the robot ensures precise handling without damaging produce.

Key Features:

  • Advanced machine vision for ripeness detection.
  • Soft gripper technology for delicate handling.
  • Automated detachment and collection mechanisms.
